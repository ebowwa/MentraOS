## Summary

Documented all message types, WebSocket flows, and HTTP endpoints for:
- Client (Glasses) ↔ Cloud protocol
- App (TPA) ↔ Cloud protocol
- Authentication flows
- Media streaming (RTMP, photos)

This specification serves as the foundation for:
- Type definitions in `@mentra/types`
- Client SDK implementation
- Protocol compliance testing
